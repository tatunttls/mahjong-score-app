4人麻雀 成績記録 v11 local sheets

開くファイル:
  index.html

今回の変更:
- 対局入力をページ式から、日付ごとのシート式に変更しました。
- 「新規対局日」ボタンから日付を選び、その日付のシートを作成できます。
- 対局日はプルダウンで切り替えられます。
- 各対局日内で「対局追加」を押すと、その日付に対局ブロックを追加します。
- 新しい対局ブロックはシート上部に表示され、古い対局は下に並びます。
- 対局日を開いたときは上部、つまり最新対局側が表示されます。
- 日付別成績のページ分けを廃止しました。
- タイトルと3つのタブは上部に固定し、下の内容だけスクロールするようにしました。

ローカル確認方法:
1. ZIPを展開します。
2. index.html をダブルクリックして開きます。
3. ブラウザだけで動作確認できます。

注意:
- firebase-config.js は同梱済みです。
- インターネット接続があり、Firebase設定が有効な場合は共有データと同期します。
- 完全にローカルだけで確認したい場合は、index.html 内の Firebase 関連 script タグ3行を一時的にコメントアウトしてください。

GitHubへ反映する場合:
- ローカル確認後、このフォルダ内の以下6ファイルをGitHubリポジトリ直下に上書きアップロードしてください。
  index.html
  style.css
  script.js
  firebase-config.js
  header-chu.png
  README.txt
